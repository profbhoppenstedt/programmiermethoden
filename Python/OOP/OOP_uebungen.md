# Übungen zu OOP in Python (Einsteiger)

## 1. Klasse `Person`

Erstelle eine Klasse `Person` mit den Attributen `name` und `age`.
Erzeuge zwei Instanzen und gib ihre Werte aus.

## 2. Methode hinzufügen

Erweitere die Klasse `Person` um eine Methode `greet()`, die eine
Begrüßung ausgibt.

## 3. Konstruktor üben

Schreibe eine Klasse `Book` mit den Attributen `title`, `author` und
`pages`. Erzeuge ein Buchobjekt.

## 4. Default-Werte

Erstelle eine Klasse `Animal` mit einem Standardwert für
`species = "unknown"`. Erzeuge mehrere Tiere.

## 5. Vererbung

Erstelle die Klasse `Vehicle` und davon abgeleitet `Car`. `Car` soll ein
zusätzliches Attribut `brand` besitzen.

## 6. Methoden überschreiben

Überschreibe in `Car` eine Methode `describe()` aus `Vehicle`.

## 7. Private Attribute

Erstelle eine Klasse `BankAccount` mit einem privaten Attribut
`__balance` und Methoden `deposit()` und `withdraw()`.

## 8. Klassenattribute

Erstelle eine Klasse `Robot` mit einem Klassenattribut `robot_count`.
Erhöhe den Zähler bei jeder Instanziierung.

## 9. Statische Methoden

Ergänze die Klasse `MathTools` um eine statische Methode `add(a, b)`.

## 10. Objekte in Listen

Erstelle mehrere Objekte der Klasse `Student` und speichere sie in einer
Liste. Gib alle Namen aus.
