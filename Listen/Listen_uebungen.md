# Übungen zu Listen in Python

## 1. Liste erstellen

Erstelle eine Liste `fruits` mit den Elementen `"Apfel"`, `"Banane"` und
`"Kirsche"`.

## 2. Element hinzufügen

Füge der Liste `fruits` das Element `"Orange"` hinzu.

## 3. Element entfernen

Entferne `"Banane"` aus der Liste `fruits`.

## 4. Länge der Liste

Gib die Anzahl der Elemente in der Liste `fruits` aus.

## 5. Zugriff auf Elemente

Gib das erste und das letzte Element der Liste `fruits` aus.

## 6. Listen durchlaufen

Schreibe eine Schleife, die alle Elemente von `fruits` ausgibt.

## 7. Elemente suchen

Prüfe, ob `"Apfel"` in der Liste `fruits` enthalten ist, und gib eine
Nachricht aus.

## 8. Liste sortieren

Sortiere die Liste `fruits` alphabetisch.

## 9. Liste umdrehen

Drehe die Reihenfolge der Liste `fruits` um.

## 10. Liste kopieren

Erstelle eine Kopie der Liste `fruits` in einer neuen Liste
`fruits_copy`.
